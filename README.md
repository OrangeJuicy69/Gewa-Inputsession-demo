🚀 Bash / Terminal Einführung (VS Code)
📌 Ziel

In diesem Projekt lernst du die Basics vom Terminal direkt in VS Code und wie du einfache Dateien erstellst und am Ende auf GitHub hochlädst.

🧰 Voraussetzungen

Bevor du startest, brauchst du:

💻 Laptop
🧑‍💻 VS Code installiert
🐙 GitHub Account
🖥️ Terminal öffnen

In VS Code:

Menü: View → Terminal
oder Shortcut: CTRL +
⚡ Erste Befehle

Diese Befehle sind die Grundlage:

pwd

👉 Zeigt dir, wo du gerade bist

ls

👉 Zeigt dir alle Dateien im Ordner

cd Desktop

👉 Wechselt in einen Ordner

📁 Projekt erstellen
mkdir mein-projekt
cd mein-projekt

👉 Erstellt einen neuen Ordner und öffnet ihn

📄 Datei erstellen
ni hallo.txt

👉 Erstellt eine Datei (PowerShell Shortcut)

Check:

ls
🌐 Auf GitHub hochladen
Repository auf GitHub erstellen
Danach im Terminal:
git init
git add .
git commit -m "erster upload"
git branch -M main
git remote add origin DEINE-URL
git push -u origin main
🔁 Workflow (merken!)
git add .
git commit -m "nachricht"
git push
🎯 Ziel erreicht

Wenn du das kannst:

✅ Terminal benutzen
✅ Dateien erstellen
✅ Projekt online hochladen

→ Dann hast du die Basics drauf 🚀
